## Agri Energy Connect

**Project Overview**
Agri Energy Connect is a web platform connecting farmers with sustainable technology providers. It streamlines access to green energy solutions and resources through a secure, role-based interface. The app is designed to accelerate renewable energy adoption in agriculture.

**Problem & Solution**

* **Problem:** Farmers lacked centralized access to technological solutions and expertise, while green energy providers struggled to reach clients.
* **Solution:** A digital ecosystem that connects farmers with tech providers, featuring product cataloguing, resource sharing, and collaboration tools.

**Key Features / Functionalities**

* ASP.NET Core MVC application with n-tier architecture
* Role-based authentication for Farmer and Employee roles using ASP.NET Identity
* CRUD operations and real-time product filtering via jQuery/AJAX
* Secure multi-format image uploads mapped to the database
* Responsive Bootstrap UI with client- and server-side validation
* Iterative development with testing, error handling, and scalable code

**Tech Stack**
ASP.NET Core MVC, Entity Framework Core, SQL Lite, Bootstrap, jQuery
